Creating a New ComputeMux Target
================================

This tutorial demonstrates how a new target can be created to target a reference
RISC-V Silicon platform, *RefSi*. It will describe how to create a new
ComputeMux target using a script, building and running a test and finally how to
modify the generated target with an additional *LLVM* wrapper pass.
